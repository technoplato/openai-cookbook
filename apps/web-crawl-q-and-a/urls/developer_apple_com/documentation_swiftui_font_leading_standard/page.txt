Font.Leading.standard
The font’s default line spacing.
iOS 14.0+
iPadOS 14.0+
macOS 11.0+
Mac Catalyst 14.0+
tvOS 14.0+
watchOS 7.0+

case standard
Discussion
If you modify a font to use a nonstandard line spacing like Font.Leading.tight or Font.Leading.loose, you can use this value to return to the font’s default line spacing.
