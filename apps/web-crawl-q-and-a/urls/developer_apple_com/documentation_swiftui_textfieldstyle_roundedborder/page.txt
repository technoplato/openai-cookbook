roundedBorder
A text field style with a system-defined rounded border.
iOS 13.0+
iPadOS 13.0+
macOS 10.15+
Mac Catalyst 13.0+
visionOS 1.0+ Beta
static var roundedBorder: RoundedBorderTextFieldStyle { get }
Available when Self is RoundedBorderTextFieldStyle.
