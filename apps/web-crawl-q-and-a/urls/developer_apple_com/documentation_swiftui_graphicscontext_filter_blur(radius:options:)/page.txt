static func blur(
    radius: CGFloat,
    options: GraphicsContext.BlurOptions = BlurOptions()

Parameters
radius
The standard deviation of the Gaussian blur.
options
A set of options controlling the application of the effect.
Return Value
A filter that applies Gaussian blur.
