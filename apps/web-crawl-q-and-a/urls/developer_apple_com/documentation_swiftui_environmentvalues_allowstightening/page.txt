Instance Property
allowsTightening
A Boolean value that indicates whether inter-character spacing should tighten to fit the text into the available space.
iOS 13.0+
iPadOS 13.0+
macOS 10.15+
Mac Catalyst 13.0+
tvOS 13.0+
watchOS 6.0+
visionOS 1.0+ Beta
var allowsTightening: Bool { get set }
Discussion
The default value is false.
