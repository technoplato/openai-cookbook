Initializer
init(color:intensity:attenuationRadius:)
Creates a point light component with the given configuration.
iOS 13.0+
iPadOS 13.0+
macOS 10.15+
Mac Catalyst 13.0+
visionOS 1.0+ Beta
init(
    color: PointLightComponent.Color = .white,
    intensity: Float = 26963.76,
    attenuationRadius: Float = 10.0
)
Parameters
color
The color of the light.
intensity
The intensity of the light in lumen.
attenuationRadius
The distance from the light, measured in meters, after which the intensity of the light becomes zero.
