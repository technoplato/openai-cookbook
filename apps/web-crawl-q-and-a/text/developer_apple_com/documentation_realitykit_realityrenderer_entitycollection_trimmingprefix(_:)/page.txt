func trimmingPrefix<Prefix>(_ prefix: Prefix) -> Self.SubSequence where Prefix : Sequence, Self.Element == Prefix.Element
Available when Element conforms to Equatable.
Parameters
predicate
A closure that takes an element of the sequence as its argument and returns a Boolean value indicating whether the element should be removed from the collection.
Return Value
A collection containing the elements of the collection that are not removed by predicate.
