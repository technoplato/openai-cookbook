type
The type of command to send the audio.
Overview
The default value is blank. Each AudioAction must specify a type.
Audio Commands
play
Begins playing the audio.
pause
Pauses the audio so a subsequent play action can resume audio from the paused position.
stop
Stops playing the audio.
Declaration
uniform token type (
    allowedTokens = ["play", "pause", "stop"]
)

The location of an audio file.
gain
A value that controls the audio volume.
auralMode
An option that controls the audio signal’s spacial dynamics.
