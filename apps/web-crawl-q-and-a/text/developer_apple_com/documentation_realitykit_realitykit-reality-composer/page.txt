Reality Composer
A visual editor for RealityKit AR scenes.
Overview
Reality Composer lets you create, compose, and edit RealityKit content visually for iOS, macOS, tvOS, and iPadOS apps. For developing visionOS apps, use Reality Composer Pro. For more information, see doc:com.apple.documentation/documentation/visionos/designing-realitykit-content-with-reality-composer-pro.
