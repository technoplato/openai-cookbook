info:id
A unique identifier for a particular type of trigger.
Overview
Triggers with different purposes define a different value for this property. For example, a runtime fires a trigger with an info:id of TapGesture when the user taps. A runtime executes a trigger with an info:id of ProximityToCamera when the user’s device crosses a distance threshhold that the trigger defines.
