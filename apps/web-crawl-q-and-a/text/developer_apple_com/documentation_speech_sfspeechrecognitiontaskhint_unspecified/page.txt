SFSpeechRecognitionTaskHint.unspecified
An unspecified type of task.

iPadOS 10.0+
macOS 10.15+
Mac Catalyst 13.1+

case unspecified = 0
Discussion
Use this hint type when the intended use for captured speech does not match the other task types.
