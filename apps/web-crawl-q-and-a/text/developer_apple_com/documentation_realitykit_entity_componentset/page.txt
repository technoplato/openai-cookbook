Entity.ComponentSet
A set of stored components.
iOS 13.0+
iPadOS 13.0+

Mac Catalyst 13.0+

@MainActor @preconcurrency
struct ComponentSet
Overview
The set represents all the components stored on an entity. It can hold only one component of any given type.
