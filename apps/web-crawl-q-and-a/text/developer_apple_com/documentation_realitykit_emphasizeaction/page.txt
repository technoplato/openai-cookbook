EmphasizeAction
An action that performs an animation to call attention to an object.
Overview
Instead of specifying your own animation, choose from the preexisting options in motionType.
Declaration
class Preliminary_Action "EmphasizeAction"
Topics
Properties
info:id
A unique identifier for the action.
affectedObjects
A list of prims that the runtime emphasizes when the action’s trigger fires.
duration
The amount of time between the start of an action and its end.
style
An option that implements different kinds of animation timing.
motionType
An option that implements animation effects.
