init(
    origin: String,
    itemCount: Int,
    totalItemCount: Int? = nil,
    metrics: [MetricsKey : Sendable]
)
Parameters
origin
A description of the event’s origin.
itemCount
The number of items processed so far.
totalItemCount
The total number of items being processed.
metrics
A dictionary of custom metrics values.
