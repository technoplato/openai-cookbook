style
An option that implements different kinds of animation timing.
Overview
The default value is “basic”.
Styles
basic
Animates with steady motion.
playful
Animates with whimsical motion.
wild
Animates with sporadic motion.
Declaration
uniform token style = "basic" (
    allowedTokens = ["basic", "playful", "wild"]
)
See Also
Properties
info:id
A unique identifier for the action.
affectedObjects
A list of prims that the runtime emphasizes when the action’s trigger fires.
duration
The amount of time between the start of an action and its end.
motionType
An option that implements animation effects.
