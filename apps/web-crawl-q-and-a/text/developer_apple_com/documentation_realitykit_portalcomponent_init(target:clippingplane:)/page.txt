Initializer
init(target:clippingPlane:)
Initializes the PortalComponent with the given target entity, and an optional clipping plane. The target entity should have a WorldComponent attached.
visionOS 1.0+ Beta
init(
    target: Entity,
    clippingPlane: PortalComponent.ClippingPlane? = nil
)
Discussion
To render a portal, an entity should have a PortalComponent and a ModelComponent, using one or more PortalMaterial instances.
