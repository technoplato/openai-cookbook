axis
A vector that describes the axis of rotation.
Overview
The object orbits in a plane perpendicular to the value of this property. To reverse an orbit direction, invert the value of this property. The default value points positively in the y-direction.
Declaration
uniform vector3d axis = (0.0, 1.0, 0.0)
See Also
Properties
info:id
A unique identifier for the action.
affectedObjects
A list of objects in orbit.
center
A prim around which the affected objects orbit.
duration
The amount of time between the start of an action and its end.
revolutions
The number of rotations to complete.
alignToPath
An option that controls the prim’s orientation as it revolves.
