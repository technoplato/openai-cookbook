locale
The locale of the speech recognizer.
iOS 10.0+
iPadOS 10.0+
macOS 10.15+
Mac Catalyst 13.1+
visionOS 1.0+ Beta
var locale: Locale { get }
Discussion
The locale of the speech recognizer is an NSLocale object. The default value of this property is the system locale (that is, system).
