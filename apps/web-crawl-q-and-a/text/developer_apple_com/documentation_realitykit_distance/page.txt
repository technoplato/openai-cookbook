distance
A threshold that measures the user’s proximity to one or more prims.
Overview
This distance is in stage units (see Encoding Stage Linear Units) and measures from the center of each prim in affectedObjects, to the device’s camera.
Declaration
uniform double distance = 0.0
See Also
Properties
info:id
A unique identifier for the trigger.
affectedObjects
A list of prims for which the runtime checks proximity.
