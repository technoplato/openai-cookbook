default
The default collision group for objects.
iOS 13.0+
iPadOS 13.0+
macOS 10.15+
Mac Catalyst 13.0+
visionOS 1.0+ Beta
static let `default`: CollisionGroup
Discussion
If no CollisionFilter is assigned to an entity, that entity will be part of this default collision group.
