yaw
The yaw angle of a face in radians.
iOS 12.0+
iPadOS 12.0+
macOS 10.14+
Mac Catalyst 13.1+
tvOS 12.0+
visionOS 1.0+ Beta
var yaw: NSNumber? { get }
Discussion
This value indicates the rotational angle of the face around the y-axis.
If the request doesn’t calculate the angle, the value is nil.
